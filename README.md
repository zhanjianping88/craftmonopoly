# Craft Monopoly

Play live: [https://craftmonopoly.com](https://craftmonopoly.com)

Craft Monopoly is a free browser game with Minecraft-inspired 3D visuals and Monopoly-style strategy gameplay. Players can buy properties, upgrade districts, pay rent, trigger chance cards, and compete against human or AI opponents.

## Play Online

Open the live site here:

- [craftmonopoly.com](https://craftmonopoly.com)

## Features

- Free browser-based gameplay
- Minecraft-style 3D board and characters
- 2 to 10 players
- Human and AI player support
- Property buying, rent, upgrades, chance cards, tax, and jail mechanics

## Local Development

To run the project locally:

```bash
python3 -m http.server 4173 --directory .
```

Then open:

- `http://localhost:4173`
